# RAPDB
The reservoir architectural parameter database
储层构型参数库存储辫状河沉积类比案例。包含地质背景和元数据表、构型单元几何参数表，两个数据表可通过“储层案例ID”关联，表中设计的数据项以理想状态下的完整信息为准.
